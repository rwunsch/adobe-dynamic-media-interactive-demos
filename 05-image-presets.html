One stable name, one place to change the rules

An Image Preset is a named bundle of rendering rules — size, quality, format, colour operations — stored centrally on the tenant. Marketers reference the short name; the expanded recipe lives in one place and can be re-tuned without touching a single page. These are real presets configured on the RobertCo tenant — no longer a concept. Pick an asset and a preset and watch the named recipe deliver live.
