Same URL, fewer bytes — automatically

Smart Imaging converts each delivery to the best modern format the visitor's browser accepts (AVIF / WebP) at an optimized quality — no URL change, no code. Below, the same request is served with Smart Imaging off (bfc=off → baseline JPEG) and on (default). Both are fetched live by your browser. Drag quality and watch the bytes on both sides move.
