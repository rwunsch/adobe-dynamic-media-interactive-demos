Why this matters

Classic Dynamic Media required publishing assets to Scene7 and pre-defining image presets. The OpenAPI /as/ endpoint is AEM-native: the approved master is the source, and any width / quality / format is computed and cached on first request.

One stable asset → infinite web-optimized renditions. Format negotiation is automatic (ask for .webp or auto and the server returns whatever best serves the requesting browser).
